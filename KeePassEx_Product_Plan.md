# KeePassEx – Production Roadmap

## Completed (MVP)
- KDBX 4.x parser
- Argon2 KDF
- AES‑CBC decryption
- XML entry extraction
- Auto‑unlock (ENV / keyfile)
- TOTP generation
- Auto‑type simulation

## To Build for "Superiority"
1. **Cross‑Platform GUI** (Flutter/Dart) – release for Windows, macOS, Linux, Android, iOS.
2. **Biometric unlock** – Windows Hello / Touch ID / Face Unlock.
3. **Hardware key support** – YubiKey HMAC‑SHA1 challenge‑response.
4. **Zero‑knowledge cloud sync** – end‑to‑end encrypted via user’s own Nextcloud or WebDAV.
5. **Browser extension** – native messaging to unlock and fill.
6. **Offline AI auto‑categorization** – local LLM to suggest entry titles.
7. **Automated backup & versioning** – git‑like history inside the KDBX.
